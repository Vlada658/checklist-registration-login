# ✅ Checklist: Registration & Login

**Описание задания:**  
Составлен комплексный чек-лист для тестирования функционала интернет-магазина, включающий ключевые пользовательские сценарии от регистрации до оформления заказа.

**Цель:**
Проверить корректность работы основных модулей интернет-магазина, включая регистрацию, авторизацию, работу с каталогом товаров, корзиной, оплатой и историей заказов

**Что было сделано:**  
- Разработан расширенный чек-лист на основе пользовательских историй для всех основных модулей интернет-магазина
- Описаны позитивные и негативные сценарии для всех ключевых функций
- Проверены валидации форм, обработка ошибок и пограничные значения
- Учтена проверка доступа к функциональности без авторизации
- Покрыт полный пользовательский путь: от входа в систему до оформления заказа
